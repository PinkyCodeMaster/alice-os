# Alice OS

Sovereign AI ecosystem that runs 100% locally.

## Features
- 100% local operation (no cloud dependencies)
- Custom voice (Deanna TTS)
- Smart home control (Matter/Thread)
- Offline capable

## Getting Started

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/alice-os.git
cd alice-os

# Install dependencies
pip install -r requirements.txt

# Run locally
python -m alice_os
```

## CI/CD

This project uses GitHub Actions for:

- Linting
- Docker builds
- Deployment to local Linux machine
